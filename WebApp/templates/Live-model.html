Facial Emotion Recognition WebApp

  1. Home
  2. Facial Emotion Recognition

Project information

  • Category: Live Deep Learning Model
  • Architecture: ResNet-50
  • Tools Used: TensorFlow and Python
  • Training Dataset: FER 2013

Requires Camera Access

The model is built on WebRTC to stream video and hence requires webcam access.

The model architecture is based on ResNet-50 and is capable of classifying among Anger, Happiness, Fear, Sadness, Surprise, Disgust, and Neutral emotions. The model was trained using the FER 2013 dataset and is deployed using TensorFlow Serving, WebRTC, Flask and uWSGI in a Linux environment.
